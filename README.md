# LLM Engineering Course

Personal exercises and projects from my LLM Engineering studies based on Ed Donner's course.

Each week contains different practical implementations and applications of Large Language Models.

## 📁 Repository Structure

### Week 2
All exercises for week 2 and 3 are located in the `week2` directory:

* **[Exercise 1 — Python Tutor Chatbot](./week2/exercise1-python-tutor)**
  <br>An interactive coding tutor chatbot built with Ollama, designed to help users learn Python with streaming responses.

* **[Exercise 2 — Synthetic Data Generator](./week2/DatasetCreationWithOllama.ipynb)**
  <br>A synthetic dataset generation tool running on Google Colab with Llama 3.1 and Gradio. It forces structured raw JSON outputs using advanced system prompts for machine learning training pipelines.
